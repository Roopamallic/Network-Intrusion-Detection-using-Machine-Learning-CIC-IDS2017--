# Network-Intrusion-Detection-using-Machine-Learning-CIC-IDS2017--
Developed an IDS using AdaBoost on the CIC-IDS2017 dataset to detect network attacks like DoS, DDoS, Brute Force, Heartbleed, Botnet, and Web Attacks. Preprocessed 80+ network features and evaluated performance using accuracy, precision, recall, and F1-score.
